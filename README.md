# C Programming Language Projects

This repository contains various programs written in C language, covering fundamental concepts, data structures, algorithms, and small projects.

## Contents

- Basic programs to practice syntax and control structures.
- Programs using arrays, strings, and pointers.
- Implementation of sorting and searching algorithms.
- Small projects such as Hotel Management System, Calculator, etc.
- Practice exercises for file handling, dynamic memory allocation, and more.

## Example Project: Hotel Management System

One of the programs is a **Hotel Management System** that allows:

- Searching hotels by city (case-insensitive).
- Taking customer details.
- Booking rooms.
- Printing reservation and payment receipts.

## How to Run Programs

1. Clone or download this repository.
2. Open a terminal/command prompt.
3. Navigate to the program folder.
4. Compile the program using `gcc`:
   ```bash
   gcc program_name.c -o program_name
